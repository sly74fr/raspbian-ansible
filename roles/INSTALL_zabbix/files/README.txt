Welcome to the Zabbix README file !

INTRO:
The purpose of the 'zbx_export_templates-LDPB_ATCA_SNMP.xml' file is to create a Zabbix template dedicated to ATCA chassis holding LDPB systems.

USAGE:
To import the template, please:
 * log to Zabbix as Admin;
 * go to Monitoring > Templates;
 * click the <Import> blue button;
 * select the 'zbx_export_templates-LDPB_ATCA_SNMP.xml' file;
 * leave all other options as is;
 * import.
 
You can then create your host with the newly imported template.
